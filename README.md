 Permissions Exercises

This repository contains a series of shell scripting exercises focused on **Linux file and directory permissions**. Each exercise is located in the `permissions/` directory, with individual scripts demonstrating different permission operations.

## Tasks Overview

1. – Switch the current user to `betty`.  
   - Command uses exactly 8 characters (plus newline).  

2.  – Prints the effective username of the current user.  

3.  – Lists all groups the current user belongs to.  

4.  – Changes the owner of the file `hello` to the user `betty`.  

5.  – Creates an empty file named `hello`.  

6.  – Adds execute permission to the owner of `hello`.  

7.  – Adds execute permission to owner and group, read permission to others for `hello`.  

8.  – Adds execute permission to owner, group, and others for `hello` (no commas allowed).  

9.  – Sets permissions for `hello` as: owner=no permission, group=no permission, others=all permissions.  

10. – Sets permissions of `hello` to `rwxr-x-wx` (owner/group/others).  

11. – Sets the permissions of `hello` the same as `olleh`.  

12. – Adds execute permissions to all **subdirectories** for owner, group, and others (files not changed).  

13. – Creates `my_dir` with permissions `751` in the current directory.  

14. – Changes the group owner of `hello` to `school`.
